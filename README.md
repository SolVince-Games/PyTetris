## Default Controls
**Left Rotate:** z\
**Right Rotate:** up\
**Move Left:** left\
**Move Right:** right\
**Soft Down:** down\
**Hard Down:** space\
**Hold:** c\
**Pause:** enter\
**Quit:** escape\
**Toggle ghost:** g\
**Toggle colour:** x\
**Scale 1:** 1\
**Scale 2:** 2\
**Scale 3:** 3\
**Scale 4:** 4\
**Volume Up:** minus\
**Volume Uown:** plus\
**Mute:** 0

## Something to note:
