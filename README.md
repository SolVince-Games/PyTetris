## Default Controls
**Left rotate:** z\
**Right rotate:** up\
**Move left:** left\
**Move right:** right\
**Soft down:** down\
**Hard down:** space\
**Hold:** c\
**Pause:** enter\
**Quit:** escape\
**Toggle ghost:** g\
**Toggle colour:** c\
**Scale 1:** 1\
**Scale 2:** 2\
**Scale 3:** 3\
**Scale 4:** 4\
**Volume up:** minus\
**Volume down:** plus\
**Mute:** 0
